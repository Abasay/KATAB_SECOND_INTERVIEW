# Rewards and Cashback History Component Documentation

## Introduction

The `RewardsAndCashbackHistory` component is a React functional component designed to display the cashbacks and miles points earned by the user. It fetches the cashbacks history data from the server and renders it in a tabular format.

## Usage

To use the `RewardsAndCashbackHistory` component, import it into your React application and include it in your component hierarchy.

```jsx
import RewardsAndCashbackHistory from "./RewardsAndCashbackHistory";

function App() {
  return (
    <div className="App">
      <RewardsAndCashbackHistory />
    </div>
  );
}
```
